# Readme

Debug snap prüft anhand von verschiedenen Paramter, wie gut, die Snap-Funktion bei der Fenstererkennung ist.

Führe aus mit:
python3 manage.py debug_snap /home/fabian/Downloads/Pläne_Fenstererkennung.pdf  --only-windows  --select nearest --ink-bg  --auto-darkness --crops

So wird unter downloads ein Snap-Ordner erzeugt, bei dem man vergleichen kann, was das Tool sieht.
